# 📚 Educational Platform

## 📖 About the Project

The **Educational Platform** is a web-based application built using **React.js**. It provides users with an interactive and engaging experience for learning, course management, and assessments with AI assistant support.

## 🚀 Features

- 📚 Course management
- 🎯 Quiz and assignment&#x20;
- 📊 Performance tracking
- 📝 AI assistant 

## 🛠 Tech Stack

- **Frontend:** React.js, HTML, CSS, JavaScript



## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- [Node.js (LTS)](https://nodejs.org/) (Recommended: **v18+**)
- [npm](https://www.npmjs.com/) 
- [Git](https://git-scm.com/)

### 📥 Installation

1️⃣ Clone the repository:

```sh
git clone https://github.com/Samcoderg788/soft-engg-project-jan-2025-se-Jan-20.git
cd soft-engg-project-jan-2025-se-Jan-20/educational-platform-frontend
```

2️⃣ Install dependencies:

```sh
npm install
```

🚀 Running the App

Start the development server:

```sh
npm start
```

The app will run at **[http://localhost:3000](http://localhost:3000)**.
