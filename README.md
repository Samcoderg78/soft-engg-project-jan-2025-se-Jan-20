# 📚 AI Agent for Academic Guidance
 
 ## 📖 About the Project
 
 The **AI Agent for Academic Guidance** is a web-based application built using **MERN Stack**. It provides users with an interactive and engaging experience for learning, course management, and assessments with AI assistant support.
 
 ## 🚀 Features
 
 - 📚 Course management
 - 🎯 Quiz and assignment&#x20;
 - 📊 Performance tracking
 - 📝 AI assistant 
 
 ## 🛠 Tech Stack
 
 - **Frontend:** React.js, HTML, CSS, JavaScript
 
 
 
 ## 🚀 Getting Started
 
 ### Prerequisites
 
 Ensure you have the following installed:
 
 - [Node.js (LTS)](https://nodejs.org/) (Recommended: **v22+**)
 - [npm](https://www.npmjs.com/) (Recommeded: **v10+**)
 - [Git](https://git-scm.com/)
 
 ### 📥 Installation
 
 1️⃣ Clone the repository:
 
 ```sh
 git clone https://github.com/Samcoderg788/soft-engg-project-jan-2025-se-Jan-20.git
 cd soft-engg-project-jan-2025-se-Jan-20/educational-platform-frontend
 ```
## Start frontend server
 
 2️⃣ Install dependencies:
 
 ```sh
 npm install
 ```
 
 🚀 Running the App
 
 Start the development server:
 
 ```sh
 npm start
 ```
## Start backend server

**Open new terminal and go to backend folder**

 ```
 cd soft-engg-project-jan-2025-se-Jan-20/educational-platform-backend
 ```
 
 The app will run at **[http://localhost:3000](http://localhost:3000)**.
